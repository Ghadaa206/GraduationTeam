Our Project

Brain Tumor

Provided to the web MR images as an input. Second step of the website
is to extract features from this input image. Texture feature extraction method is used for
extracting features from the MR image. Now the tumor was detected.
After feature extraction, these features independently are used for
classification as malignant and benign MR image.
Ensemble base classifier is used for classification.
It classifies the brain image on the bases of multiple classifiers.
No more processing is required once the MR image is determined as benign.
But when the MR image is determined as malignant by the classifier
it is further processed for extracting tumor portion from it.
For this purpose segmentation is performed on this MR image.
our objective is detecting the disease and helps in the prediction at an early stage
where the brain tumor can be avoided and kept to be a benign one.

( Detection , Classification and Segmentation)
